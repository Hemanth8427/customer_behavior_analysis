# customer_behavior_analysis
Data anlytics project showcasing customer behavior analysis using python, SQL and power Bi

📌 Overview
This project analyzes 3,900 customer transactions to understand shopping behavior, spending patterns, product preferences, and subscription trends.
The workflow covers:
- Loading and cleaning data in Python
- Performing Exploratory Data Analysis (EDA)
- Running business‑focused SQL queries in MySQL/PostgreSQL
- Building an interactive Power BI dashboard
- Creating a final report and presentation using Gamma
The goal is to generate insights that support data‑driven decision‑making for retail businesses.

📁 Dataset
The dataset contains 18 columns and 3,900 rows, including:
- Customer Demographics: Age, Gender, Location, Subscription Status
- Purchase Details: Item Purchased, Category, Amount, Season, Size, Color
- Behavioral Metrics: Discount Applied, Previous Purchases, Frequency, Review Rating
- Shipping Details: Shipping Type
Example from the document:
“This project analyzes customer shopping behavior using transactional data from 3,900 purchases across various product categories.”

🔄 Project Steps
1. Data Loading (Python)
- Imported dataset using pandas.read_csv()
- Checked structure using .info() and .describe()
- Identified missing values (e.g., 37 missing review ratings)
2. Data Cleaning & Preparation
- Imputed missing review ratings using median per product category
- Standardized column names to snake_case
- Created new features:
- age_group (binned age ranges)
- purchase_frequency_days
- Removed redundant columns (e.g., promo_code_used)
From the document:
“Verified if discount_applied and promo_code_used were redundant; dropped promo_code_used.”
3. Exploratory Data Analysis (EDA)
- Distribution of age, purchase amount, ratings
- Category‑wise sales and revenue
- Discount usage patterns
- Customer segmentation (New, Returning, Loyal)
4. SQL Analysis (MySQL/PostgreSQL)
Executed business queries such as:
- Revenue by gender
- High‑spending discount users
- Top‑rated products
- Shipping type comparison
- Subscriber vs non‑subscriber revenue
- Discount‑dependent products
- Top 3 products per category
- Repeat buyers vs subscription likelihood
Example from the document:
“Compared total revenue generated by male vs. female customers.”
5. Power BI Dashboard
Built an interactive dashboard showing:
- Subscription breakdown
- Revenue & sales by category
- Revenue by age group
- Shipping type comparison
- Customer demographics
- Average purchase amount & review rating
6. Final Report & Presentation
- A written report summarizing insights and recommendations
- A clean, modern Gamma presentation for stakeholders

📈 Key Insights & Results
Some insights derived from the analysis:
- Male customers generated higher revenue than female customers.
- Hat, Sneakers, and Coat were the most discount‑dependent products.
- Young Adults contributed the highest revenue.
- Express shipping users spent slightly more on average.
- Loyal customers formed the largest segment (3,116 customers).
From the document:
“Young Adult customers contributed $62,143 in total revenue.”
